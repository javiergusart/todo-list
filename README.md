# My Todos

A simple todo list app built with React and Vite. It displays a list of tasks so you can keep track of what you need to get done.

## Installation

Make sure you have Node.js installed, then clone the repo and install the dependencies.

```
npm install
```

## Running the Dev Server

```
npm run dev
```

Then open your browser and go to the local address shown in the terminal, usually http://localhost:5173.
